# SmartQB-QT Ultimate Edition

🚀 A powerful, AI-driven, Fluent Design desktop application for exam generation and parsing.

## Features

- **Modern UI**: PySide6 + qfluentwidgets for an immersive Windows 11 experience.
- **AI Integration**: Compatible with OpenAI APIs for text correction, tagging, and generation.
- **Fast Vector Search**: Full integration with LanceDB.
- **Background Parsing**: CPU-optimized PP-StructureV3 runs asynchronously for non-blocking UI.
- **Secure Setup**: Local SQLite database specifically for configuration management with AES-256-GCM encryption.
